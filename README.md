# **R1614's** Server Introduction

Welcome to the documentation for **R1614's Server**. 

## Storage Server Specifications

### Hardware
- **Processor**: N4000 (2 cores, 2 threads)
- **Memory**: 6GB RAM
- **Storage**:
  - Seagate 500GB HDD
  - Western Digital 500GB HDD
  - 128GB EMMC (OS)

### Operating System
- **OpenMediaVault 7.0**

## Docker Containers

- **NginxProxyManager-Plus**
- **Filebrowser**
- **Dozzle**
- **Dockge**
- **Voltaserve**
- **Watchtower**
- **Homarr**
- **Homepage**
- **Dashdot**
- **Excalidraw**

#### One-command install
```bash
wget https://raw.githubusercontent.com/r1614/setup/main/full.sh
chmod +x full.sh
./full.sh
```
